Math & Statistical Formulas
01
SUM Math
Adds all numbers in a range or list of values
▼

SUM adds together all numbers. You can provide individual cells, ranges, or a mix. One of Excel's most fundamental functions.

Syntax
=SUM(number1, [number2], ...)
Parameters
number1Required. First number, cell, or range to add.
number2…Optional. Up to 255 additional numbers/ranges.
Examples
FormulaDescriptionResult
=SUM(10, 20, 30)Direct values60
=SUM(A1:A10)Sum a rangeSum of A1–A10
=SUM(A1:A5, C1:C5)Multiple rangesBoth ranges combined
=SUM(A1:A10)-B1Sum minus a valueNet total
Keyboard Shortcut
Alt+=Auto-sum selected range
💡 Tip: Select a cell below/beside a column and press Alt+= to instantly insert =SUM() with the range auto-detected.

Lookup & Reference Functions
01
VLOOKUP Lookup
Searches a column and returns a value from another column
▼

VLOOKUP (Vertical Lookup) is one of the most used Excel functions. It searches down the first column of a table for a match, then returns a value from a specified column.

=VLOOKUP(lookup_value, table_array, col_index_num, [range_lookup])
lookup_valueThe value to search for (e.g., employee ID, product code)
table_arrayThe range containing the lookup table (lookup col must be FIRST)
col_index_numColumn number to return (1 = first column of table_array)
range_lookupFALSE = exact match (recommended); TRUE = approximate match
FormulaDescription
=VLOOKUP(A2, $E$2:$G$100, 2, FALSE)Exact match — find A2 in col E, return col F
=VLOOKUP(A2, Sheet2!$A:$C, 3, 0)Lookup across sheets
=IFERROR(VLOOKUP(A2,B:D,2,0),"Not Found")Error-safe VLOOKUP
=VLOOKUP(A2&B2, C:E, 3, 0)Concatenate two lookup values
=SUM(VLOOKUP(A2,B:G,{3,4,5},0))VLOOKUP with SUM (multiple columns)
⚠️ Common Errors:
• #N/A — value not found (check exact spelling, use TRIM)
• #REF! — col_index exceeds table width
• Wrong values — forgot FALSE/0 for exact match
• Slow on large data — lookup column not locked with $
💡 Modern Excel: Replace VLOOKUP with XLOOKUP (Excel 2021+) — no column number needed, supports reverse lookup, and returns arrays: =XLOOKUP(A2, B:B, C:C, "Not Found")
02
VLOOKUP with MATCH Lookup
Dynamic column number — no hardcoding col index
▼

Replace the fixed col_index_num with a MATCH function that finds the column number dynamically based on a header.

=VLOOKUP(lookup, table, MATCH(col_header, header_row, 0), 0)
FormulaDescription
=VLOOKUP(A2, $B$1:$F$100, MATCH("Salary",$B$1:$F$1,0), 0)Find column by header name dynamically
💡 Benefit: If columns are inserted or rearranged, this formula auto-adjusts. Pure VLOOKUP col_index breaks when you insert columns!
03
INDEX with MATCH Lookup
More flexible alternative to VLOOKUP — lookup any direction
▼

INDEX returns a value at a given row/column position. MATCH finds the position of a value in a range. Together they form the most powerful lookup combination.

=INDEX(return_range, MATCH(lookup_value, lookup_range, 0))
FormulaDescription
=INDEX(C:C, MATCH(A2, B:B, 0))Basic INDEX/MATCH — find A2 in B, return C
=INDEX(A:A, MATCH(E2, C:C, 0))Reverse lookup — look left (impossible with VLOOKUP!)
=INDEX(B:B, MATCH(1,(A:A=D2)*(C:C=E2),0))Two-criteria lookup (Ctrl+Shift+Enter)
📌 INDEX/MATCH Advantages over VLOOKUP:
✅ Can look left (return columns to the LEFT of lookup)
✅ Column position doesn't break when inserting columns
✅ Faster on large datasets
✅ Not limited to first column
04
HLOOKUP Lookup
Horizontal lookup — searches across a row
▼
=HLOOKUP(lookup_value, table_array, row_index_num, [range_lookup])
FormulaDescription
=HLOOKUP("Q2", $A$1:$D$5, 3, FALSE)Find "Q2" in row 1, return row 3 value
=HLOOKUP(A1, B1:F4, 2, 0)Return 2nd row value
📌 HLOOKUP searches horizontally (rows). VLOOKUP searches vertically (columns). Use when your lookup table is laid out in rows.
05
OFFSET with MATCH Lookup

=OFFSET(A1, MATCH(E1,A:A,0)-1, 2)Find row of E1 in A, offset 2 cols right

Financial Functions
01
PMT Financial
Calculates the periodic payment for a loan or investment
▼
=PMT(rate, nper, pv, [fv], [type])
rateInterest rate per period (annual rate ÷ 12 for monthly)
nperTotal number of payment periods
pvPresent value (loan amount, enter as negative)
fvFuture value (default 0 for fully paid loan)
type0 = end of period; 1 = beginning of period
FormulaScenarioResult
=PMT(8%/12, 5*12, -500000)₹5L loan @ 8%/yr, 5 yearsMonthly payment
=PMT(10%/12, 240, -3000000)₹30L home loan, 20 yearsEMI
📌 PMT returns a negative number (cash outflow). Use -PMT(...) or enter pv as positive to get positive EMI.
02
IPMT Financial
Returns the interest portion of a loan payment
▼
=IPMT(rate, per, nper, pv, [fv], [type])
FormulaResult
=IPMT(10%/12, 1, 120, -1000000)Interest in payment #1
=IPMT(10%/12, 60, 120, -1000000)Interest in payment #60 (midpoint)
💡 Interest decreases over time as principal is repaid. Use IPMT down the rows (per=1,2,3…) to build a full amortization schedule.
03
PPMT Financial
Returns the principal portion of a loan payment
▼
=PPMT(rate, per, nper, pv, [fv], [type])
FormulaResult
=PPMT(10%/12, 1, 120, -1000000)Principal repaid in payment #1
📌 Relationship: PMT = IPMT + PPMT. Verify: each payment's interest + principal should equal the fixed EMI.
04
PV Financial
Present Value — what future cash flows are worth today
▼
=PV(rate, nper, pmt, [fv], [type])
FormulaScenarioResult
=PV(8%/12, 60, -5000)Worth of ₹5000/mo for 5 yrs @ 8%Present value
💡 Use Case: Is a lease worth it? Use PV to compare lump-sum vs recurring payments. If PV > upfront cost, payments are better.
05
FV Financial
Future Value — how much an investment grows over time
▼
=FV(rate, nper, pmt, [pv], [type])
FormulaScenarioResult
=FV(12%/12, 240, -5000)₹5000/mo SIP for 20 yrs @ 12%Corpus at maturity
=FV(7%, 10, 0, -100000)₹1L one-time @ 7% for 10 yrs196,715
💡 SIP Calculator: =FV(12%/12, 240, -5000, 0, 1) — type=1 means SIP paid at START of each month, giving slightly better returns.

MS Excel — Features & UI Guide
01
Excel Introduction & File Menu
▼

Excel is Microsoft's spreadsheet application used for data organization, analysis, calculations, and visualization. The File menu (Backstage View) manages workbook operations.

  • New — Create a blank workbook or from a template
  • Open — Open existing .xlsx, .xls, .csv, .xlsm files
  • Save / Save As — Save current file or save with new name/format
  • Print — Preview and print with settings for margins, orientation, scaling
  • Share — Share via OneDrive, email attachment, or link
  • Export — Convert to PDF, CSV, or other formats
  • Options — Configure Excel behavior, add-ins, formulas, language
Essential File Shortcuts

  • Borders — Add cell borders: outline, inside, thick, dashed, colored

04
Conditional Formatting
▼

Apply formatting automatically based on cell values or formulas. Turns your data into a visual dashboard without charts.

  • Highlight Cell Rules — Greater than, less than, equal to, between, text containing, duplicate values
  • Top/Bottom Rules — Top 10 items, top 10%, bottom 10, above average, below average
  • Data Bars — Horizontal bar inside cells proportional to value (in-cell chart)
  • Color Scales — 2- or 3-color gradient across a range (heat map effect)
  • Icon Sets — Arrows, traffic lights, stars, check marks based on value thresholds
  • New Rule (Custom Formula) — Use formulas like =MOD(ROW(),2)=0 to stripe alternate rows
  • Manage Rules — View, edit, reorder, and delete existing rules
💡 Pro Tips:
• Use =$A1="Pending" (lock column, not row) to highlight entire rows
• Stack multiple rules — first TRUE rule wins (manage order carefully)
• Use ISBLANK(), ISNUMBER(), ISTEXT() in custom formulas
05
Format as Table & Cell Styles
▼
  • Format as Table — Converts a range to a structured table with auto-filter, banded rows, total row
  • Structured References — Formulas use column names like =[@Sales]*[@Rate] instead of cell addresses
  • Auto-Expand — Adding data below/beside a table automatically includes it in formulas and ranges
  • Total Row — Toggle a summary row with dropdown for Sum/Average/Count/Min/Max
  • Cell Styles — Pre-built formatting combinations: heading, title, accent colors, calculation styles
  • Keyboard: Ctrl+T to create a table from selection
Ctrl+TCreate Table
💡 Tables are the #1 Excel best practice — all formulas referencing a table auto-update when data grows!

  • Fill Down/Right/Series — Ctrl+D fills down, Ctrl+R fills right; Fill → Series for number patterns
  • Flash Fill — Ctrl+E intelligently fills patterns detected from examples (extract first name, format phone, etc.)
  • Clear Contents — Delete key clears content only; Clear All removes content AND formatting
  • Sort A→Z / Z→A — Quick single-column sort; Data → Sort for multi-level sorting
  • Filter — Ctrl+Shift+L toggles AutoFilter; dropdown on each column header to filter by value, color, condition

09
Table & Pivot Table
▼

Pivot Tables are Excel's most powerful data analysis tool. They allow you to summarize, group, and cross-tabulate large datasets in seconds.

  • Create Pivot Table — Insert → Pivot Table; choose source data and placement
  • Field Areas — Drag fields to: Filters (report filter), Columns (cross-tab), Rows (groups), Values (aggregate)
  • Value Settings — Right-click → Value Field Settings: Sum, Count, Average, Max, Min, % of total, etc.
  • Group — Group dates by month/quarter/year; group numbers into bins
  • Calculated Field — Create custom formulas within the Pivot Table (e.g., Profit = Revenue - Cost)
  • Refresh — Alt+F5 or right-click → Refresh after source data changes
  • Pivot Chart — Insert a chart linked to a Pivot Table; updates with filters automatically
  • Slicers — Visual filter buttons; click to filter Pivot Table without using dropdown arrows
Alt+F5Refresh Pivot Ctrl+TCreate Table
💡 Best Practice: Convert source data to a Table (Ctrl+T) before creating a Pivot — the Pivot will auto-include new rows when you refresh!

  • Area — Like line but fills below; shows cumulative value
  • Combo Chart — Mix bar and line on same chart with secondary axis
  • Waterfall — Show sequential addition/subtraction (perfect for P&L)
  • Histogram — Distribution of data in bins
  • Chart Elements — Add/remove: title, axis labels, gridlines, data labels, legend, trendline
